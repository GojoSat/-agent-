
# 🚀 Multi-Agent Automation System

## 📌 项目简介
这是一个基于大语言模型的多 Agent 协同自动化系统，实现任务拆解与自动执行。

## 🧠 系统架构
User Input → Task Manager → Research Agent → Writer Agent → Optimizer Agent → Output

## ⚙️ 功能
- 多 Agent 协同
- 自动内容生成
- 可扩展 Agent
- 简单任务调度

## 🚀 快速开始

```bash
pip install openai
python main.py
```

## 🔑 配置
在 main.py 中填写你的 API Key：
```
client = OpenAI(api_key="YOUR_API_KEY")
```
